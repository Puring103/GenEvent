# 项目描述

GenEvent 是一个高性能,0GC的Event库
使用代码生成器实现,无运行时反射

# 主要特性

1. 无运行时反射
   使用编辑器代码生成，避免运行时反射开销

2. 0 GC
   使用值类型，无额外开销

3. 高易用性的流式api

4. IL2cpp友好
   消除了运行时反射代码，可以在IL2CPP中正常运行

5. 支持事件优先级，事件拦截，共享事件与事件过滤
6. 确定性的执行优先级
7. 最小化生成代码

# 实现

1. 为Event和Subscriber生成扩展类
2. 优先级直接在生成时决定
3. 拦截根据返回值决定
4. 事件过滤通过传入过滤器实现

# 注意

1. 不需要修改unity文件夹下内容，这里的代码是自动生成的
